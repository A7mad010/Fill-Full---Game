# Fill Full Game System

## Overview
Wall Filling is a puzzle based game system where the player must fill holes in walls using interactive buttons, Each wall presents a unique pattern that challenges the player’s observation and timing skills.

## Gameplay
The player faces multiple walls containing missing blocks.  
By clicking buttons, blocks are spawned and move toward the wall to fill empty spaces, Completing a wall allows progression to the next challenge.

## System Features
- Modular wall generation
- Dynamic hole patterns
- Button based interaction
- Automatic hole detection
- Wall completion validation
- Scalable difficulty

## Tech
- Unity Engine
- C#

## How to Play
1. Observe the wall and locate the holes.
2. Click buttons to spawn blocks.
3. Fill all empty spaces.
4. Complete the wall to progress.
